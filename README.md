shootmap
========

AngularJS / Ionic mobile application

Run these commands:

(only if you don't have cordova globally installed)
```
npm install -g cordova
```
(only if you don't have ionic globally installed)
```
npm install -g ionic
```
```
git clone https://github.com/dotGiff/shootmap.git
```
```
cd shootmap
```
```
ionic serve 
```
That's all!
